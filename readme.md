# 🎯PLAYTASK

一个轻量级、易部署的个人习惯养成与任务管理工具，通过目标追踪 + 积分激励的核心机制，帮助你持续养成好习惯、高效完成任务。

## ✨ 核心功能:

- 📋 任务 / 习惯创建与管理
- 📊 目标完成进度追踪
- 🏆 积分激励体系（完成任务获取积分）
- 🔄 数据实时同步（基于 Supabase）

## 前端

- **原生HTML/CSS/JS**，保持轻量，避免框架依赖
- 轻量级、高性能、易扩展
- 响应式设计（适配多端）

## 后端

**Supabase**，免费额度充足，PostgreSQL强大，实时功能优秀

## 📋 环境要求

### 运行环境

- 现代浏览器（Chrome 90+ / Firefox 88+ / Safari 14+）
- 网络环境（需连接 Supabase 服务））

### 开发环境

- Node.js 16+（仅用于开发辅助工具，如代码格式化）
- Supabase CLI（可选，用于本地开发调试）
